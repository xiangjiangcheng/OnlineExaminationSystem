# OnlineExaminationSystem

## 工程实践2 - 在线考试系统
    在线考试系统是为了减轻教师出题、改卷负担，同时也便于学生自我测试。
    主要针对试卷中的客观题而开发的系统。

## 开发环境
    * IDE: Intellij IDEA 14
    * Java: JDK 7 + Tomcat 7 or higher
    * Build Tool: Maven 3
    * Backend: Spring MVC + Spring + Spring Data JPA
    * Web: Bootstrap 3 + jQuery
    * Database: MySQL 5.6
